# AP Luxury WordPress Theme Installation

## 1. Download the theme

From GitHub, click **Code → Download ZIP**.

## 2. Upload to WordPress

In WordPress Admin:

1. Go to **Appearance → Themes**.
2. Click **Add New**.
3. Click **Upload Theme**.
4. Upload the ZIP file.
5. Activate **AP Luxury**.

## 3. Create pages

Create these WordPress pages and assign the matching template from the Page settings sidebar:

- Home — use the site front page
- About — Template: About
- Services — Template: Services & Pricing
- Gallery — Template: Gallery
- Booking — Template: Booking
- Promotions — Template: Promotions & Membership
- Testimonials — Template: Testimonials & Reviews
- Contact — Template: Contact

## 4. Set the homepage

Go to **Settings → Reading**:

- Your homepage displays: **A static page**
- Homepage: **Home**

## 5. Set menus

Go to **Appearance → Menus** and create a menu with:

- Home
- About
- Services
- Gallery
- Promotions
- Testimonials
- Contact
- Booking

Assign it to **Primary Menu**.

Create a smaller footer menu and assign it to **Footer Menu**.

## 6. Add your logo

Go to **Appearance → Customize → Site Identity** and upload the AP's Thread Salon logo.

## 7. Update salon information

Go to **Appearance → Customize → AP Luxury Business Info** and update:

- Phone number
- Address
- Business hours
- Booking URL
- Instagram URL
- Google Map text/embed placeholder

## 8. Booking widget

Open the Booking page and paste your booking provider code or shortcode into the page content area.

Recommended providers:

- Square Appointments
- Vagaro
- Fresha
- Booksy

## 9. Recommended plugins

- Rank Math SEO
- WPForms
- Smash Balloon Instagram Feed
- Google Reviews plugin
- LiteSpeed Cache or WP Rocket
- Imagify or ShortPixel

## 10. Final setup notes

Replace placeholder gradients with real salon photos when available. Use compressed WebP images for best mobile speed.
