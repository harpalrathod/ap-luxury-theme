# AP Luxury Theme

A custom luxury WordPress theme for **AP's Thread Salon**, a premium eyebrow threading and beauty salon in Rockwall, TX.

## Brand Direction

- Minimal luxury aesthetic
- Emerald and gold palette
- Serif headline typography
- Mobile-first booking experience
- Local SEO focused for threading, waxing, facials, aftercare, and beauty services

## Included Templates

- Homepage
- About
- Services & Pricing
- Aftercare Guide
- Gallery
- Booking
- Promotions & Membership
- Testimonials & Reviews
- FAQ
- Service Area
- Setup Guide
- Privacy Policy
- Terms of Service
- Contact
- Blog archive
- Single post
- Search
- 404
- WooCommerce shop wrapper

## Features

- Sticky luxury header
- Floating Book Now button
- Responsive navigation
- Scroll reveal animations
- WordPress Customizer business settings
- Custom homepage copy settings
- First-visit offer popup
- Newsletter and contact form placeholders
- Instagram and Google Reviews placeholders
- Local BeautySalon schema
- WooCommerce-ready
- Elementor-friendly
- SEO-focused page structure
- Salon shortcodes
- Accessibility and print CSS support

## Shortcodes

- `[ap_booking_button]`
- `[ap_phone]`
- `[ap_address]`
- `[ap_offer]`
