# AP Luxury Quick Launch Steps

Use this quick version when you are ready to put the AP Luxury theme on the live website.

## 1. Install Theme

1. Download the repository ZIP from GitHub.
2. Upload it in WordPress under Appearance → Themes → Add New → Upload Theme.
3. Activate AP Luxury.

## 2. Create Core Pages

Create the main pages and assign matching templates:

- Home
- About
- Services
- Aftercare
- Gallery
- Booking
- Promotions
- Testimonials
- FAQ
- Service Area
- Contact
- Privacy Policy
- Terms

## 3. Set Homepage

Go to Settings → Reading and set Home as the static homepage.

## 4. Add Business Details

Go to Appearance → Customize and update:

- Logo
- Phone
- Address
- Hours
- Booking link
- Instagram link
- Map details
- Homepage hero text
- First-visit offer text

## 5. Connect Plugins

Connect these before launch:

- Booking provider
- Contact form plugin
- Google Reviews plugin
- Instagram feed plugin
- SEO plugin
- Cache plugin

## 6. Replace Placeholders

Replace all placeholder image blocks with real salon images. Check services, prices, promotions, privacy text, and terms text.

## 7. Final Test

Test mobile menu, Book Now buttons, phone links, contact page, booking page, popup offer, reviews area, and footer links.
